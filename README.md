# El Meridiano — Diario Digital

Portada de diario digital desarrollada como **Trabajo Práctico N.º 1 – Flexbox** para la materia Desarrollo y Arquitectura Web (DAW) — Universidad Abierta Interamericana (UAI).


## 📋 Descripción

Portada original de diario online llamado **El Meridiano**, construida con HTML semántico y CSS con Flexbox como herramienta principal de layout. Diseño responsive con enfoque Mobile First.

## 🎨 Decisiones de diseño

- **Paleta editorial**: tonos cálidos (crema, sepia, tinta) para evocar la estética de diario impreso trasladada al digital.
- **Tipografía**: `Playfair Display` (titulares) + `Source Serif 4` (cuerpo) + `DM Sans` (UI/etiquetas).
- **Color de acento**: rojo oscuro (`#B22222`) para elementos de énfasis y categorías de política; azul marino (`#1E3A5F`) para categorías secundarias.
- **Layout**: columnas con Flexbox, sin CSS Grid ni frameworks externos.
